Create Entity Procedure

The createEntity procedure type can create other entities as a procedure step. This is useful when views or similar need to be created as part of a procedure running.

The createEntity procedure type can create all kinds of entities: procedures, functions, datasets and plugins.

Execution

When run, the procedure will have the same effect as a POST of the given object to /v1/<kind>s endpoint.

Errors

In the case of an error, the run will indicate that the step had an error.

Configuration

A new procedure of type createEntity named <id> can be created as follows:

mldb.put("/v1/procedures/"+<id>, {
    "type": "createEntity",
    "params": {
        "id": <string>,
        "type": <string>,
        "params": <Any>,
        "persistent": <bool>,
        "deterministic": <bool>,
        "kind": <string>,
        "runOnCreation": <bool>
    }
})

with the following key-value definitions for params:

Field, Type, DefaultDescription

id
string

ID of the entity. This indicates what name the entity will be called when accessed via the REST API.

type
string

Type of the entity. This indicates what implementation type to use for the entity.

params
Any

Entity configuration parameters. This is always an object, and the type of object accepted depends upon the 'type' field.

persistent
bool
false

If true, then this element will have its configuration stored and will be reloaded on startup

deterministic
bool
true

If true, the entity has no hidden state

kind
string

Kind of entity to create. Valid values are 'function', 'procedure', 'dataset' or 'plugin'.

runOnCreation
bool
true

If true, the procedure will be run immediately. The response will contain an extra field called firstRun pointing to the URL of the run.

Output

The procedure's output will be in the following form, which allows for the expanded configuration and the ID of the new entity to be obtained by the caller:

Field, Type, DefaultDescription

kind
string

Kind of entity created

config
PolyConfig

Configuration of the entity created

status
Any

Status of entity creation
