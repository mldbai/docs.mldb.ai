Mock Procedure

Mock procedure that fakes progressing on something for a configurable amount of time. Useful to test progression and cancellation.

Configuration

A new procedure of type mock named <id> can be created as follows:

mldb.put("/v1/procedures/"+<id>, {
    "type": "mock",
    "params": {
        "durationMs": <int>,
        "refreshRateMs": <int>,
        "runOnCreation": <bool>
    }
})

with the following key-value definitions for params:

Field, Type, DefaultDescription

durationMs
int
5000

Total duration in ms the procedure will fake work.

refreshRateMs
int
50

The frequency at which the procedure will update its fake progress

runOnCreation
bool
true

If true, the procedure will be run immediately. The response will contain an extra field called firstRun pointing to the URL of the run.
