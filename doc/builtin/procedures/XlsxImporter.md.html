Xlsx importer (experimental)

This procedure will import all worksheets from a .xlsx (not .xls) workbook into an MLDB dataset.

Configuration

A new procedure of type experimental.import.xlsx named <id> can be created as follows:

mldb.put("/v1/procedures/"+<id>, {
    "type": "experimental.import.xlsx",
    "params": {
        "runOnCreation": <bool>,
        "dataFileUrl": <Url>,
        "output": <OutputDatasetSpec>
    }
})

with the following key-value definitions for params:

Field, Type, DefaultDescription

runOnCreation
bool
true

If true, the procedure will be run immediately. The response will contain an extra field called firstRun pointing to the URL of the run.

dataFileUrl
Url

URL to load Excel workbook from

output
OutputDatasetSpec

Configuration for output dataset

Output

Each cell will be converted to its normal data type. The row names are in the format Sheet:cell, for example Sheet1:01 for the first row. The column names are the Excel column IDs, for example A for the first column and AA for the 26th column.
