Legacy Lookalikes Function

This feature is part of the MLDB Pro Plugin and so can only be used in compliance with the trial license unless a commercial license has been purchased

This function is used to load legacy mldb.ai lookalike models into MLDB. Please contact mldb.ai for documentation and contract terms if you are a mldb.ai Audience Optimizer customer.

Configuration

A new function of type legacy.lal named <id> can be created as follows:

mldb.put("/v1/functions/"+<id>, {
    "type": "legacy.lal",
    "params": {
        "svdModelFileUrl": <Url>,
        "classifierModelFileUrl": <Url>
    }
})

with the following key-value definitions for params:

Field, Type, DefaultDescription

svdModelFileUrl
Url

URL to load the legacy SVD model from (normally a svd.bin.gz file)

classifierModelFileUrl
Url

URL to load the legacy model from (normally a .bin.gz file)
