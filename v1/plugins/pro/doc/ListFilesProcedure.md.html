List Files Procedure

This procedure recursively lists files found under a path and stores their information to the output dataset. File names will be used as row names. The columns of the dataset are: * size * lastmodified * etag * storageclass * ownerid * ownername

Configuration

A new procedure of type list.files named <id> can be created as follows:

mldb.put("/v1/procedures/"+<id>, {
    "type": "list.files",
    "params": {
        "path": <Url>,
        "modifiedSince": <Date>,
        "outputDataset": <OutputDatasetSpec>,
        "maxDepth": <int>,
        "runOnCreation": <bool>
    }
})

with the following key-value definitions for params:

Field, Type, DefaultDescription

path
Url

The path to start listing files from.

modifiedSince
Date
"-Inf"

Filter that will keep the files modified since the specified date.

outputDataset
OutputDatasetSpec
{"type":"tabular"}

Output dataset configuration. This may refer either to an existing dataset, or a fully specified but non-existing dataset which will be created by the procedure.

maxDepth
int
-1

Maximum depth of directories to go in. -1 means unlimited.

runOnCreation
bool
true

If true, the procedure will be run immediately. The response will contain an extra field called firstRun pointing to the URL of the run.
